## 🚀 启动步骤

### 方法一：命令行启动（推荐）

1. **打开终端/命令提示符**
   - Windows: 按 `Win + R`，输入 `cmd` 或 `powershell`
   - 或在文件夹中右键 → "在终端中打开"

2. **进入前端目录**
   ```bash
   cd C:\Users\Administrator\Desktop\文献脚注参考翻译pro\Auto_cite_V2\frontend
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **等待启动完成**
   - 看到类似以下输出表示成功：
   ```
   VITE v6.4.1  ready in 231 ms

   ➜  Local:   http://localhost:5174/
   ➜  Network: http://10.161.10.251:5174/
   ```

5. **打开浏览器访问**
   - 在浏览器中打开：`http://localhost:5174`
   - 或按住 `Ctrl` 点击终端中的链接

---

## ⚠️ 常见问题

### 1. 端口被占用
**现象**: 提示 "Port 5173 is in use"
**解决**: Vite会自动尝试下一个端口（5174、5175等），无需手动处理

### 2. 依赖缺失
**现象**: 启动时报错找不到某个模块
**解决**:
```bash
npm install
```

### 3. 浏览器显示空白
**现象**: 页面加载但显示空白
**解决**:
- 打开浏览器开发者工具（F12）查看控制台错误
- 检查是否有JavaScript错误
- 尝试清除浏览器缓存后刷新

### 4. 热更新不工作
**现象**: 修改代码后页面不自动刷新
**解决**:
- 手动刷新浏览器（Ctrl + R）
- 重启开发服务器（Ctrl + C 停止，然后重新运行 npm run dev）
